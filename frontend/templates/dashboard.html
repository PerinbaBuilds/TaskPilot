Tier Routing Explanation
Why tier pools? Without pools, the most powerful server wins every metric after normalisation — performance, latency, and often cost too. Tier pools enforce sustainability: 🌿 green jobs run on efficient small servers (lowest CO₂ + cost), ⚡ performance jobs get powerful large servers, and ⚖️ balanced jobs use mid-tier hardware. Each RL agent then fine-tunes the decision within its pool based on real-time carbon and load signals.
